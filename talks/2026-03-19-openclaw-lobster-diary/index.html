🏠 內部風險

龍蝦自己出事

  • 過度授權
    Agent 自己花 $3,000 買課程（真實案例！）

    API Key 被讀走

    龍蝦說的不一定對，要驗證

🌐 外部風險

外面打進來

  • Prompt Injection
    惡意網頁騙龍蝦做壞事
  • 模型 DoS
    被灌爆 context，龍蝦當機
  • 不安全輸出
    龍蝦輸出直接被執行
